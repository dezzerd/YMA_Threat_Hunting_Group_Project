# data-raw

Скрипты в этой папке предназначены для разработки и воспроизводимого запуска
ETL-процесса (скачивание → приведение данных → сохранение в файл/БД).

В R-пакет они **не включаются** (см. `.Rbuildignore`), но служат как
документированный вычислительный процесс для команды.

Рекомендуемый старт:

1. Открыть `data-raw/01_dbip_build_artifacts.R`
2. Запустить скрипт (при необходимости — с `version = "latest"` или конкретным `YYYY-MM`)
